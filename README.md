# 🛠️ TextUtils - Universal Text Processing

A high-performance, 100% client-side text utilities website. Process your text securely without ever leaving your browser.

## 🚀 Features

- **56+ Tools**: Word counters, case converters, JSON formatters, encoders, and more.
- **Privacy First**: All processing happens locally in your browser. No data is sent to any server.
- **Lightning Fast**: No server roundtrips means instant results.
- **SEO Optimized**: Each tool has its own dedicated, search-engine-friendly URL.
- **Dark Mode**: Toggle between light and dark themes for comfortable use.

## 🛠️ Tech Stack

- **HTML5**: Semantic structure for accessibility and SEO.
- **Tailwind CSS**: Modern, responsive styling.
- **Vanilla JavaScript**: Fast, dependency-free logic.
- **GitHub Actions**: Automated deployment to GitHub Pages.

## 💻 Getting Started

To run this project locally, ensure you have [Node.js](https://nodejs.org/) installed, then follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/viveknimbolkar/textutils.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Build the project**:
   ```bash
   npm run build
   ```
4. **Start the local server**:
   ```bash
   npm run serve
   ```
   The site will be available at `http://localhost:3000`.

## 📖 How to Use

1. Navigate to the homepage to see all available tools.
2. Use the **Search Bar** (Cmd+K) to find a specific tool instantly.
3. Paste your text into the input area.
4. The results are processed in real-time or via the action buttons.
5. Copy or download your processed text using the provided buttons.

## 📝 Adding New Tools

1. Copy an existing tool file (e.g., `word-counter.html`) to `new-tool.html`.
2. Update the `<title>`, `<meta description>`, and `<h1>`.
3. Create a corresponding logic file in `js/modules/new-tool.js`.
4. Add the new tool entry to `js/data.js` to make it searchable.
5. Add a link in `index.html` and `sitemap.html`.
